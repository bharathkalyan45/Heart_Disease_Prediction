# ❤️ Heart Disease Prediction

## 📌 Overview

This project is a **Heart Disease Prediction System** that uses machine learning algorithms to predict whether a person is at risk of heart disease based on clinical and lifestyle parameters. The goal is to assist in early detection and preventive healthcare decision-making.

---

## 🎯 Objectives

* Predict the presence of heart disease accurately
* Analyze patient health parameters using ML models
* Support early diagnosis and risk assessment
* Build a simple and reliable prediction system

---

## 🧠 Technologies Used

* **Python** – core programming language
* **Scikit-learn** – machine learning models
* **Pandas & NumPy** – data preprocessing and analysis
* **Matplotlib / Seaborn** – data visualization
* **FastAPI / Flask** – backend deployment
* **HTML, CSS, JavaScript** – frontend interface

---

## 🗂️ Dataset

* Medical dataset containing patient health information
* Common features include:

  * Age
  * Sex
  * Chest pain type
  * Resting blood pressure
  * Cholesterol level
  * Fasting blood sugar
  * ECG results
  * Maximum heart rate
  * Exercise-induced angina

*(Dataset sourced from public repositories such as UCI / Kaggle)*

---

## ⚙️ Workflow

1. User enters patient health parameters
2. Data preprocessing and feature scaling
3. Input passed to trained ML model
4. Model predicts heart disease risk
5. Prediction result is displayed

---

## 🚀 Features

* Binary classification (Disease / No Disease)
* Fast and accurate predictions
* User-friendly medical input form
* Deployable as a web application

## ▶️ How to Run the Project

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/heart-disease-prediction.git
cd heart-disease-prediction
```

### 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Run the Application

```bash
python app.py
```

OR (FastAPI)

```bash
uvicorn main:app --reload
```

### 4️⃣ Open in Browser

```
http://127.0.0.1:8000
```
